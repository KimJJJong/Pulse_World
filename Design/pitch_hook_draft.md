# Pulse World — Hook 전략서

> 콘텐츠(게임 체험) Hook과 사업(문서/투자) Hook을 분리하여 정리

---

# Part 1. 🎮 콘텐츠 Hook — "플레이어가 빠져드는 장치"

> 게임 자체의 매력. 플레이어가 처음 접했을 때 이탈하지 않고 빠져들게 만드는 요소.

## 1-1. 감각 Hook — "이 세계는 살아있다"

**타이밍:** 게임 시작 0~5초

플레이어가 아무 버튼도 누르지 않았는데 **세계가 이미 뛰고 있다.**

- 타이틀 로고가 맥동에 맞춰 확장/수축
- 배경의 땅, 나무, 조명이 한 리듬에 동기화
- 음악이 배경이 아니라 **세계의 물리법칙**

> 💡 "뭐지 이 게임?" — 기존 리듬 게임과 다르다는 걸 **말 없이** 보여줌

## 1-2. 체험 Hook — "무기를 들자마자 달라진다"

**타이밍:** 5~30초 (튜토리얼 대체)

| 순서 | 장면 | 플레이어 반응 |
|------|------|--------------|
| ① | 맨손. 맥동은 느끼지만 아무것도 못 함 | "뭘 하지?" |
| ② | 바닥의 검을 줍자 맥류가 흘러들며 이펙트 폭발 | "오!" |
| ③ | 비트에 맞춘 공격 → 강한 참격 + 맥류 폭발 | "비트에 맞추면 강해지는구나" |
| ④ | 다른 무기(활, 지팡이)를 줍자 전혀 다른 공격 | "무기가 능력이구나!" |

> 💡 설명 없이 체험으로 핵심 메카닉 전달. "무기를 들면 내가 바뀐다" = 수집 욕구 자극

## 1-3. 서사 Hook — "왜 이 세계가 뛰는가?"

**타이밍:** 30초~ (답을 주지 않고 질문만 던짐)

| Hook 질문 | 암시 | 유발 감정 |
|-----------|------|----------|
| "심장은 왜 뛰는가?" | 멈추면 세계 소멸, 그런데 점점 빨라지고 있다 | 불안 + 호기심 |
| "맥류 심핵은 뭘 원하는가?" | 보스가 오히려 뭔가를 막고 있는 건 아닌지 | 반전 기대 |
| "BPM이 계속 올라가면?" | 심연 끝에 뭐가 있나 | 도전 욕구 |

## 1-4. 시스템 Hook — "한 번 더 해야 하는 이유"

| 장치 | 설명 |
|------|------|
| **무기 × 맥석 조합** | 같은 검이라도 맥석에 따라 화염/빙결/관통 → 빌드 실험 욕구 |
| **BPM 에스컬레이션** | 깊이 갈수록 빨라짐 → "내 한계는 어디까지?" |
| **레이드 보스 박자 변조** | 4/4 → 3/4로 바뀌는 순간의 긴장 → 클리어 시 성취감 |
| **멀티 협력** | 파티원과 맥동 동기화 → 혼자와 다른 재미 |

## 1-5. 추천 오프닝 시퀀스

```
1. 완전한 정적. 화면 어둡고 소리 없음.
2. "...쿵" — 첫 맥동. 화면이 미약하게 빛남.
3. "...쿵...쿵" — 맥동 반복, 세계가 밝아짐.
4. BPM 안정 → 세계 전체 깨어남, BGM 시작.
5. 바닥에 무기 하나. 줍는 순간 맥류 폭발 → 전투 시작.
```

# Part 2. 📄 사업 Hook — "심사위원이 넘기지 않는 장치"

> 지원 사업 서류·IR·사업계획서 초두에서 평가자의 시선을 잡는 전략.

## 2-1. 오프닝 한 줄 — "읽는 순간 멈추게 하기"

세 가지 톤 중 서류 성격에 맞게 선택:

| 톤 | 오프닝 문장 | 적합한 곳 |
|----|-----------|----------|
| **도발형** | "리듬 게임에서 '음악'을 빼면 뭐가 남을까요?" | 문체부·콘텐츠진흥 |
| **정의형** | "무기를 바꾸면 리듬이 바뀌는 세계" | 데모데이·IR 피칭 |
| **공백형** | "리듬 × RPG × 멀티의 교차점, 아직 아무도 없습니다" | 창업·투자·진흥원 |

## 2-2. 시장 공백 어필 — "왜 지금, 왜 이것인가"

```
글로벌 리듬 게임 시장: ~$2.8B (2024), 연 8.2% 성장
그러나 95% 이상이 [단곡 플레이 → 점수 경쟁] 구조
```

**공백:** 리듬의 긴장감 + RPG의 성장감 + 멀티플레이의 사회성을 **동시에** 제공하는 PC 타이틀이 없음

## 2-3. 경쟁작 비교 — "우리만 할 수 있는 것"

| | 크립트 오브 네크로댄서 | 하이-파이 러시 | **Pulse World** |
|------|---------------------|---------------|-----------------|
| 리듬 연동 깊이 | 이동만 비트 제한 | 전투 보너스 | **모든 행동이 맥동 의존** |
| 성장 시스템 | 로그라이크 (초기화) | 고정 스킬셋 | **무기·맥석 영구 성장** |
| 멀티플레이 | ❌ | ❌ | **✅ 1~4인 실시간 협력** |
| 보스 기믹 | 없음 | 연출 위주 | **BPM·박자 변조** |
| 빌드 다양성 | 아이템 조합 (일회성) | 없음 | **무기 × 맥석 = 무한 빌드** |

## 2-4. 차별점 3줄 요약 — "핵심만 남기기"

심사위원이 요약란만 읽어도 매력을 느끼도록:

> **① 무기가 곧 능력** — 스킬 트리 없이 무기 교체만으로 전투 스타일 전환
> **② 세계가 뛴다** — 리듬이 BGM이 아닌 물리법칙. 맞추면 공명, 빗나가면 실패
> **③ 보스가 규칙을 바꾼다** — 전투 중 BPM·박자 변조로 플레이어 적응력 시험

## 2-5. 사업성 어필 포인트

| 항목 | 포인트 |
|------|--------|
| **수익 모델** | 패키지 판매 + 유료 확장팩 (부분 유료화·시즌패스 없음) |
| **리텐션 구조** | 무기 수집 + 맥석 조합 + 레이드 도전 → 장기 플레이 유도 |
| **확장성** | 무기·맥석·던전 DLC 추가만으로 후속 매출 → 낮은 콘텐츠 추가 비용 |
| **타겟** | PC 코어 게이머 (리듬 게임 경험 불필요, RPG 팬 유입 가능) |
| **글로벌 가능성** | 리듬은 언어 장벽 없음. 로컬라이징 비용 최소 |
