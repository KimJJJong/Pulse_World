# 🌍 맥동하는 세계 (Pulse World)

> *이 세계의 모든 물질과 에너지는 '대지의 맥동(Pulse)'에 의해 작동한다.*

---

# 📌 핵심 개념

## 대지의 심장 (The Great Heart)

세계 지하 최심부에 잠든 존재. 끊임없이 맥동하며 **맥류(Pulse Flow)**를 만든다.

- 모든 생명과 마법은 **맥류**에 의존
- 맥동의 **정점(Peak)**에서만 에너지가 최대로 응축
- 정점 사이 = **공허(Dead Interval)** — 에너지 극도로 희박

## 왜 비트에 맞춰야 하는가?

> 💡 행동(전투, 마법, 이동)은 에너지를 소모한다. 맥동의 정점에 맞춰야만 맥류에서 에너지를 끌어쓸 수 있다.

| 판정 | 세계관 | 효과 |
|------|--------|------|
| Perfect | 완전 공명 | 최대 위력 |
| Good | 부분 공명 | 감쇠된 위력 |
| Miss | 공허 | 에너지 끌어오지 못함 → 행동 실패 |

# 🪝 시작 Hook — 첫 인상 장치

## Layer 1. 시각적 Hook (0~5초) — "이 세계는 살아있다"

> 💡 타이틀/로딩 화면부터 화면 전체가 심장처럼 뛴다. 플레이어가 아무것도 하지 않아도 **세계 자체가 이미 리듬을 치고 있다.**

- 로고 자체가 맥동에 맞춰 미세하게 확장/수축
- 배경의 땅, 나무, 건물 조명까지 모든 것이 **한 리듬에 동기화**
- BGM이 아니라 **세계 자체가 음악** → "뭐지 이 게임?" 유발

## Layer 2. 체험적 Hook (5~30초) — "무기를 들자마자 달라진다"

> 💡 튜토리얼 대신 무기를 주워보게 한다. 설명 없이 체험으로 이해시킨다.

| 순서 | 장면 | 플레이어 반응 |
|------|------|--------------|
| ① | 맨손으로 서있음. 맥동은 느끼지만 아무것도 못 함 | "뭘 하지?" |
| ② | 바닥에 떨어진 검을 줍는다 → 맥류가 검으로 흘러들며 이펙트 폭발 | "오!" |
| ③ | 비트에 맞춰 공격 → 강한 참격이 맥류와 함께 터짐 | "비트에 맞추면 강해지는구나" |
| ④ | 다른 무기(활, 지팡이)를 줍는다 → 완전히 다른 공격 방식 | "무기가 능력이구나!" |

**핵심:** "스킬을 배운다" 가 아니라 **"무기를 들면 내가 바뀐다"** → 무기 수집 욕구 즉시 자극

## Layer 3. 서사적 Hook (30초~) — "왜 이 세계가 뛰는가?"

> 💡 답을 주지 않고, **질문만 던져서** 끌어당긴다.

| Hook 질문 | 서사 암시 | 플레이어 호기심 |
|-----------|----------|----------------|
| "대지의 심장은 왜 뛰는가?" | 심장이 멈추면 세계 소멸... 하지만 점점 빨라지고 있다 | "무슨 일이 일어나는 거지?" |
| "맥류 심핵은 뭘 원하는가?" | 레이드 보스 = 심장에서 떨어진 존재. 뭔가를 막으려는 걸까? | "적이 세계를 지키는 건 아닌가?" |
| "BPM이 계속 올라가면?" | 심연으로 갈수록 맥동 가속 = 심장이 폭주 중 | "끝에 뭐가 있는 거지?" |

## 🎬 추천 오프닝 시퀀스: "침묵 → 첫 맥동"

```
1. 완전한 정적 — 화면 어둡고 소리 없음
2. "...쿵" — 첫 맥동. 화면이 미약하게 빛남
3. "...쿵...쿵" — 맥동이 반복되며 세계가 밝아짐
4. 맥동이 안정 BPM에 도달 → 세계 전체가 깨어남, BGM 시작
5. 바닥에 무기 하나 → 줍는 순간 맥류 폭발 → 전투 시작
```

> 💡 "세계가 눈앞에서 켜지는" 경험. 말 없이 보여주는 것이 가장 강력한 Hook.

# 👤 캐릭터

## 맥류사 (Pulse Dancer)

맥동을 읽고 그 리듬에 맞춰 싸우는 전사.

- 일반인은 맥동을 못 느끼지만, 맥류사는 **맥동감(Pulse Sense)**이 각성된 사람
- 리듬감이 뛰어난 사람 = 맥류 적성 높음
- 맥류사 자체는 "직업"이 아님 — **무기가 능력을 결정**
- 같은 맥류사라도 **어떤 무기를 드느냐**에 따라 전투 스타일이 완전히 달라짐

# 👾 몬스터 — 맥류 변이체

맥류가 과잉 집중된 곳에서 물질이 자아를 가지게 된 존재. 본능적으로 맥동에 동기화되어 **항상 비트에 맞춰 행동**. 처치 시 맥석 드롭.

## 등급 체계

### 🟢 일반 — 맥류 잔향 (Pulse Echo)

- 잔여 파동이 물질에 깃든 **약한 변이체**
- 자아가 희미, 단순 본능으로 움직임
- **단순 패턴**, 낮은 HP
- 솔로/듀오 처리 가능

### 🟡 엘리트 — 맥류 결정체 (Pulse Crystalline)

- 고밀도 맥류가 **결정화**된 강력한 변이체
- 뚜렷한 자아, 복잡한 맥동 패턴 구사
- **복합 패턴 + 페이즈 전환**
- 파티 협력 권장

### 🔴 레이드 — 맥류 심핵 (Pulse Core)

- 대지의 심장에서 **직접 분리**된 거대 존재
- 자체적으로 **맥동 주파수를 왜곡** 가능 → BPM 변동
- **BPM 변동 · 박자 변조(3/4, 5/4) · 맵 변형 · 다단 페이즈**
- 다수 파티 필요. 최상위 보상
- 처치 시 **고순도 맥석** 드롭 → 상위 장비 핵심 재료

# ⚔️ 시스템 ↔ 세계관 매핑

## 리듬 시스템

| 코드 | 세계관 |
|------|--------|
| `Bpm` | 맥동 주파수. 지역 깊이에 따라 결정 |
| `BaseBeatDivision` | 맥동 세분도. 강력한 맥류 지역은 복잡하게 갈라짐 |
| `ActionWindowMs` | 맥동 정점의 에너지 집중 시간 |
| `OnBeat` | 맥동 정점 도달 — 세계가 미세하게 진동 |

## 무기 시스템

> 💡 무기 = 맥석(Pulse Crystal)이 내장된 무구. 맥석의 종류와 배치에 따라 맥류를 다른 형태로 변환한다.

### 핵심 개념

- **무기가 곧 능력** — 별도의 스킬 트리 없이, 무기 자체가 공격 방식·범위·특수 효과를 결정
- 같은 "대검"이라도 내장된 **맥석 조합**에 따라 화염형, 빙결형, 관통형 등으로 분기
- 무기를 바꾸면 전투 스타일 자체가 변함

### 무기 액션 ↔ 세계관

| 액션 | 세계관 |
|------|--------|
| **Warning** | 무기의 맥석이 맥류를 수렴 → 대지가 반응 (전조 현상) |
| **Damage** | 무기가 증폭한 맥류를 폭발적으로 방출 |
| **Move** | 무기-맥석의 공명으로 맥류 흐름을 타고 이동 |
| **InputLock** | 무기에 잔류 맥류가 남아 안정화까지 행동 불가 (에너지 반동) |
| `WarningColorStep` | 무기 맥석의 수렴 강도 변화 (노랑→빨강) |

## 몬스터 패턴

> 💡 몬스터는 태어날 때부터 맥류의 일부. 인간과 달리 훈련 없이 본능적으로 동기화.

| 요소 | 세계관 |
|------|--------|
| **PatternSet (페이즈)** | 맥류 공명 강도 변화. HP↓ → 불안정 → **과부하로 더 거친 패턴** |
| **WeightedPick** | 맥류가 유도하는 본능적 행동 선택 |
| **Cooldown** | 맥류 패턴 에너지 재충전 |
| **TargetDef** | 맥류를 통한 적 감지. 근접할수록 교란이 커서 감지 용이 |

## 스테이지 / 맵

| 요소 | 세계관 |
|------|--------|
| **Stage** | 맥류 구역 — 심장에서 뻗어나가는 에너지 지대 |
| **MapId** | 맥류가 만든 지형 유형 (동굴, 유적, 결정지대) |
| `StartDelayMs` | 조율(Tuning) 시간 — 현지 맥동에 감각을 맞추는 준비 |
| MonsterAllDead | 맥류 오염원 제거 완료 |
| OpenGate | 맥류 봉인 해제 → 새 통로 개방 |
| SpawnMonster | 맥류 교란 → 새 변이체 현현 |
| ReturnToTown | 맥류 정화 완료 → 안전지대로 맥류 역류(귀환) |

## BPM 증가 / 무한 모드

| 요소 | 세계관 |
|------|--------|
| 레벨↑ → BPM↑ | 더 깊은 곳 = 심장에 가까움 = 맥동이 더 격렬 |
| **Infinite Mode** | **심연의 시련** — 심장 가장 깊은 곳, 끊임없이 빨라지는 맥동 |
| `mBpm` | 층 하강 시 맥동이 한 단계씩 격해짐 |
| `MaxBpm` | 인간이 감내할 수 있는 맥동 공명 한계치 |

# ⚙️ 장비 & 강화

## 무기 = 핵심 성장 축

| 시스템 | 세계관 |
|--------|--------|
| **무기** | 맥석 내장 무구. 무기 종류 × 맥석 조합 = **능력 결정** |
| **방어구** | 맥류 차폐/흡수 장구. 방어 성능에 영향 |
| **강화** | 맥석 공명률을 높이는 조율 작업 → 무기 위력 증가 |
| **맥석 교체** | 무기에 내장된 맥석을 바꿔 능력 특성 변경 |
| **칼리브레이션** | 맥류사가 자신의 맥동감을 환경에 미세 조율하는 의식 |

> 💡 무기를 바꾸면 전투 스타일이 바뀐다. 무기 수집 & 맥석 조합이 캐릭터 성장의 핵심.

# 🎨 3D 비주얼 연출

## 맥동 표현 (매 비트)

- 바닥 셰이더: 중앙에서 원형 파동 (Ripple Shader)
- 환경광: 미세하게 밝아짐
- 바닥 그리드: 순간적으로 빛남
- 파티클: 바닥에서 에너지 입자 솟아오름

## 판정별 이펙트

| 판정 | 연출 |
|------|------|
| Perfect | 강한 에너지 오라 폭발 + 바닥 균열 빛 + 카메라 줌 |
| Good | 약한 에너지 오라 + 빛 산란 |
| Miss | 에너지 흩어짐 파티클 + 캐릭터 흔들림 |

## Warning 전조 연출

1. **Beat N** — 대상 셀 바닥에 균열 생성, 약한 빛 누출
2. **Beat N+1** — 균열 확대, 빛 강도 증가, 미세 진동
3. **Beat N+2** — 빛 최대, 바닥 부풀어오름, 경고음
4. **Damage** — 맥류 폭발! 바닥 파괴 + 충격파

## 몬스터 맥류 표현

- **일반** — 미약한 맥류 발광, 비트마다 눈/코어 깜빡
- **공격 준비** — 맥류 빛 증가 + 공기 왜곡
- **페이즈 전환** — 과부하 균열/빛 누출, 이펙트 격렬화

## InputLock (경직)

- 시작: 잔류 에너지 파티클 + 캐릭터 회색빛
- 종료: 에너지 회복 이펙트 (밝아지며 파티클 흡수)
